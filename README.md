### 🛍 Customer Segmentation — K-Means · Hierarchical · DBSCAN + Supervised Classifier

**Tools:** `Python` `Scikit-learn` `XGBoost` `Pandas` `Seaborn` `Streamlit` · **Date:** 2026.02 · Individual Project
Segmented 200 mall customers using three unsupervised clustering algorithms, then built and tuned a supervised classifier to predict segments for new customers — deployed as an interactive web dashboard.

- Compared K-Means, Hierarchical Clustering, and DBSCAN using Silhouette, Davies-Bouldin, and Calinski-Harabasz metrics — K-Means achieved the best Silhouette Score (0.55)
- Applied feature scaling (StandardScaler) and used Elbow Method + Silhouette analysis to determine optimal k = 5
- Trained and tuned 4 classifiers (Logistic Regression, Random Forest, XGBoost, SVM) with GridSearchCV + 5-fold stratified CV — SVM achieved best CV accuracy (97.5%)
- Identified 5 actionable segments: Cautious Shoppers, Impulsive Buyers, Average Customers, Wealthy but Frugal, and VIP Targets
- Deployed a full Streamlit dashboard with segment prediction, cluster analytics, model comparison, and business strategy recommendations

🔗 [GitHub Repo](https://github.com/kanlayakornchanthanam/Customer_Segmentation)
